# GeneratePress Child Clean

Tema hijo básico para **GeneratePress** con estructura mínima y genérica.

## Incluye

- `functions.php` con carga simple de módulos.
- `inc/setup.php` para soportes y menús.
- `inc/enqueue.php` para CSS/JS del child theme.
- `assets/css/theme.css` con estilos base de ejemplo.
- `assets/js/theme.js` con interacción básica de ejemplo.
- `single.php` y `404.php` limpios.

## Uso

1. Activa el tema hijo en WordPress.
2. Edita `assets/css/theme.css` para tu diseño.
3. Edita `assets/js/theme.js` para interacciones personalizadas.
